Required components:
Python:
Python is the language used for this project. It can be downloaded form the official website: https://www.python.org/downloads/
Select the correct OS and click the download python.
To check if python was correctly installed, open the command prompt and type:  Python. This command will return the version installed.

Numpy:
“NumPy is the fundamental package for scientific computing in Python. It is a Python library that provides a multidimensional array object, various derived objects (such as masked arrays and matrices), and an assortment of routines for fast operations on arrays, including mathematical, logical, shape manipulation, sorting, selecting, I/O, discrete Fourier transforms, basic linear algebra, basic statistical operations, random simulation and much more.” -https://numpy.org/doc/stable/user/whatisnumpy.html
Download procedure (https://numpy.org/install/):
Can be downloaded from source or using pip:
With pip: in the terminal, run the command pip install numpy

Note: In case pip is not installed on the user’s machine, follow this guide: https://pip.pypa.io/en/stable/installation/


Installation and run procedure:
-Download the BSP4-MatLab folder
-make sure Python is installed
-make sure Numpy is installed
-open the folder in the command prompt 
-enter the command  python senza44.py  to run the code
-put the xls file that will be used in the correct directory
-type the name of the xls file that will be used ex: TFM2.xls
